# ApportionmentCalc
Kalkulator alokasi kursi pemilu: Sainte-Laguë vs Kuota Hare.

## Pendahuluan
ApportionmentCalc adalah alat yang dirancang untuk membantu pengguna menghitung dan memvisualisasikan alokasi kursi pemilu. Alat ini membandingkan dua metode populer: **metode Sainte-Laguë** (metode Webster) dan **metode Kuota Hare** (metode sisa suara terbanyak). Alat ini sangat berguna untuk memahami bagaimana sistem pemilu yang berbeda menerjemahkan suara menjadi kursi parlemen.

## Fitur
- **Perbandingan Metode**: Perbandingan berdampingan alokasi kursi menggunakan metode Sainte-Laguë dan Kuota Hare.
- **Langkah Detail**: Lihat perhitungan langkah demi langkah untuk metode Sainte-Laguë, termasuk pembagi dan bilangan (quotient) untuk setiap putaran.
- **Input yang Dapat Disesuaikan**: Sesuaikan jumlah total kursi dan persentase ambang batas parlemen dengan mudah.
- **Manajemen Data**:
  - **Impor**: Muat data suara dari file CSV atau TSV.
  - **Ekspor**: Unduh hasil perhitungan sebagai file CSV.
- **Dukungan Multi-bahasa**: Tersedia sepenuhnya dalam Bahasa Inggris dan Bahasa Indonesia.
- **Dukungan Tema**: Pilih antara tema Terang, Gelap, atau Otomatis (sesuai sistem).
- **Privasi Utama**: Semua perhitungan dilakukan secara lokal di peramban Anda; tidak ada data yang meninggalkan perangkat Anda.
- **Satu Berkas**: Berkas HTML mandiri yang ringan tanpa dependensi eksternal.

## Penggunaan
1. Unduh berkas `PreferenceRank.html`.
2. Buka di peramban web modern apa pun.
3. Masukkan jumlah total kursi yang akan dialokasikan.
4. Atur ambang batas parlemen (jika ada).
5. Tambahkan partai dan jumlah suara mereka, atau impor dataset.
6. Klik **Hitung Alokasi** (Calculate Allocation) untuk melihat hasilnya.
